## ðŸŽ® Game Rules
- The game is played on a **3Ã—3 grid**.
- Two players take turns:
  - Player 1 uses **X**
  - Player 2 uses **O**
- A player wins by placing **three of their marks in a row**:
  - Horizontally  
  - Vertically  
  - Diagonally
- If all cells are filled and no player wins, the game results in a **draw**.
